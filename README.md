# 🚀 CareerForge

🌐 Live Demo: https://careerforge01.vercel.app/

CareerForge is an AI-powered career development platform that helps students and job seekers create ATS-friendly resumes, analyze resume strength, improve job readiness, and receive intelligent career guidance.

---

## ✨ Features

- 📄 Resume Analysis
- 🎯 ATS Score Evaluation
- 🤖 AI-Powered Suggestions
- 💼 Career Guidance
- 📊 Interactive Dashboard
- 📱 Responsive Design
- ⚡ Fast and User-Friendly Interface

---

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript
- Python
- Flask

- NumPy

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/Harshaaa0203/CAREERFORGE.git
cd CAREERFORGE
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Application

```bash
python app.py
```

Open:

```bash
http://127.0.0.1:5000
```
